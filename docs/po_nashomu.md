# 3D Finite Element Method (FEM) Solver & Visualizer

Програмний комплекс для розрахунку напружено-деформованого стану 3D-тіл за допомогою Методів Скінченних Елементів (МСЕ). Проєкт написаний на C з використанням графічної бібліотеки **Raylib** для рендерингу сітки, деформацій та теплової карти напружень у реальному часі.

## Математична модель та алгоритми

### 1. Дискретизація та ізопараметрична апроксимація

Область розбивається на $nel$ скінченних елементів. Для апроксимації переміщень всередині елемента використовуються ізопараметричні функції форми $\phi_i(\alpha, \beta, \gamma)$ у локальній системі координат $\alpha, \beta, \gamma \in [-1, 1]$.

Переміщення $u(x, y, z)$ у будь-якій точці елемента визначається як:


$$u(x, y, z) = \sum_{i=1}^{20} \phi_i(\alpha, \beta, \gamma) u_i$$


де $u_i$ — переміщення у вузлах скінченного елемента.

Оскільки елемент ізопараметричний, глобальні координати також виражаються через функції форми:


$$x = \sum_{i=1}^{20} \phi_i(\alpha, \beta, \gamma) x_i, \quad y = \sum_{i=1}^{20} \phi_i(\alpha, \beta, \gamma) y_i, \quad z = \sum_{i=1}^{20} \phi_i(\alpha, \beta, \gamma) z_i$$

### 2. Матриця жорсткості елемента (MGE)

Локальна матриця жорсткості розміром 60x60 обчислюється з використанням матриці Якобі $[DJ]$ для переходу від глобальних до локальних координат:


$$[DJ] = \begin{bmatrix} \frac{\partial x}{\partial \alpha} & \frac{\partial y}{\partial \alpha} & \frac{\partial z}{\partial \alpha} \\ \frac{\partial x}{\partial \beta} & \frac{\partial y}{\partial \beta} & \frac{\partial z}{\partial \beta} \\ \frac{\partial x}{\partial \gamma} & \frac{\partial y}{\partial \gamma} & \frac{\partial z}{\partial \gamma} \end{bmatrix}$$

Для обчислення матриці жорсткості $K^e$ застосовується 3D інтегрування Гауса (по 3 точки на кожну вісь, всього 27 точок):


$$K^e = \sum_{m=1}^3 \sum_{n=1}^3 \sum_{k=1}^3 w_m w_n w_k B^T D B \det(J_{mnk})$$


де $w$ — вагові коефіцієнти Гауса, $D$ — матриця пружності (залежить від модуля Юнга $E$ та коефіцієнта Пуассона $\nu$), а $B$ — матриця похідних функцій форми по глобальних координатах.

Коефіцієнти Ламе обчислюються як:


$$\lambda = \frac{E \cdot \nu}{(1+\nu)(1-2\nu)}, \quad \mu = \frac{E}{2(1+\nu)}$$

### 3. Вектор навантаження (FE)

Прикладення тиску $P$ до грані елемента моделюється поверхневим інтегралом. Використовуються 2D функції форми $\psi_i(\eta, \tau)$ для 8 вузлів грані:


$$F^e = \iint_S N^T P \, dS$$

Обчислення виконується 2D квадратурами Гауса (3x3 = 9 точок):


$$F^e = \sum_{m=1}^3 \sum_{n=1}^3 w_m w_n P (J_{2D}) \psi_i(\eta, \tau)$$


де нормаль та площа враховуються через векторний добуток дотичних векторів на поверхні $\left(\frac{\partial \mathbf{r}}{\partial \tau} \times \frac{\partial \mathbf{r}}{\partial \eta}\right)$.

### 4. Граничні умови (ZU)

Для врахування жорсткого закріплення (переміщення $u_x=0, u_y=0, u_z=0$) використовується **метод штрафу**. На головну діагональ глобальної матриці жорсткості $[MG]$ у відповідних вузлах додається велике число (штраф), а у вектор навантаження записується 0:


$$MG[i][i] = 10^{16}, \quad F[i] = 0$$

### 5. Розв'язання глобальної СЛАР

Зібрана глобальна матриця жорсткості $[MG]$ та вектор навантажень $F$ утворюють систему:


$$[MG] \{U\} = \{F\}$$


Оскільки матриця $[MG]$ є симетричною і додатно визначеною, для пошуку вектора переміщень $\{U\}$ використовується ітераційний **Метод Спряжених Градієнтів (Conjugate Gradient Method)**, що забезпечує швидку збіжність для розріджених систем.

### 6. Розрахунок напружень (Von Mises)

Після знаходження переміщень обчислюються компоненти тензора деформацій:


$$\epsilon_{xx} = \frac{\partial u_x}{\partial x}, \quad \epsilon_{yy} = \frac{\partial u_y}{\partial y}, \quad \gamma_{xy} = \frac{\partial u_x}{\partial y} + \frac{\partial u_y}{\partial x}$$

Тензор напружень за законом Гука:


$$\sigma_{xx} = \lambda(\epsilon_{xx} + \epsilon_{yy} + \epsilon_{zz}) + 2\mu\epsilon_{xx}$$

$$\tau_{xy} = \mu \gamma_{xy}$$

Еквівалентне напруження за критерієм фон Мізеса (Von Mises) обчислюється для візуалізації теплової карти:


$$\sigma_{v} = \sqrt{\frac{1}{2} \left[ (\sigma_{xx}-\sigma_{yy})^2 + (\sigma_{yy}-\sigma_{zz})^2 + (\sigma_{zz}-\sigma_{xx})^2 + 6(\tau_{xy}^2 + \tau_{xz}^2 + \tau_{yz}^2) \right]}$$
